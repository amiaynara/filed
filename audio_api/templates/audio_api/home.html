Api Documentation

Following are the available endpoints
  • http://localhost:8000/ GET
  • http://localhost:8000/<file_type>/ GET POST
  • http://localhost:8000/<file_type>/<id>/ GET PUT DELETE
Available file types:
  • song
  • podcast
  • audiobook
Example Api
http://localhost:8000/song/
